# 你画我猜

基于微信云开发的多人在线你画我猜小程序。

## 技术栈

- 前端：微信小程序原生框架（WXML / WXSS / JavaScript）
- 后端：微信云函数（Node.js + `wx-server-sdk`）
- 数据：微信云数据库
- 同步：`watch` + 轮询兜底

## 主要功能

- 登录并保存用户信息
- 创建房间 / 输入房间号加入房间
- 等待页准备、房主开局
- 画手出题、作画、清空画板
- 猜手猜词与计分
- 回合轮换、结算弹窗、排行榜展示

## 游戏流程

1. 登录并创建或加入房间
2. 等待页准备，房主开始游戏
3. 画手输入词条后进入作画阶段
4. 猜手猜词，系统同步分数与状态
5. 回合结束后轮换画手
6. 达到总轮数后显示结算

## 部署

1. 用微信开发者工具导入项目
2. 开通云开发环境
3. 创建集合：`users`、`room`、`rounds`
4. 上传 `cloudfunctions/` 下的云函数
5. 编译运行

## 安全规则

开发阶段可先将 `room` 和 `rounds` 集合设为：

```json
{
  "read": true,
  "write": true
}
```

否则非创建者可能无法正常收到房间同步数据。
